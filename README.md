# Netchoose

A simple GUI to switch between [netctl][0] profiles, written with Rust and GTK.

Current status: Alpha, not really user friendly yet.

![screenshot](https://rawgit.com/dbrgn/netchoose-rs/master/screenshot.png)

## Requirements

- netctl
- gksudo
- GTK 3.6+
